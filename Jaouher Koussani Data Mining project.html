I.Introduction¶

The dataset used in this project is the car evaluation dataset, which we downloaded from [the UCI directory] https://archive.ics.uci.edu/ml/datasets/Car+Evaluation it is a dataframe with 6 features, 1 class variable and 1728 instances.

The objective of this project is to build and compare models which can predict if a car is acceptable or not, based on 6 features.

Filter model¶

We will try to find the best features of our dataset using a minimum redundance, maximum relevance approach. For this purpose we will test if there's a significant correlation between each independant variable (X) and the dependant variable (Y). Then we will check if there is a mutual information between independant variables. For all of these tests we will use the chi square test since all of our variables are categorical

    classifier = KNeighborsClassifier(n_neighbors = i)
    classifier.fit(X_train, y_train)
    y_train_pred = classifier.predict(X_train)
    y_test_pred  = classifier.predict(X_test)
    
    acc_train[i]=metrics.accuracy_score(y_train,y_train_pred)
    acc_test[i]=metrics.accuracy_score(y_test_pred, y_test)
    num_neighbors[i]=i
    
plt.title('The classification Accuracy')

plt.legend()
